# 🛡️ About ModBot

ModBot is a Discord moderation bot built to help servers stay safe, organized, and easy to manage.

## 📌 What it does
- 🔨 Moderation tools (ban, kick, mute, warn)
- 🧾 Full warning & mute tracking system
- 📊 Automatic moderation logs per server
- 📝 Private mod notes for staff
- ⚙️ Custom server settings (prefix, modlog channel)
- 📡 Command usage tracking for transparency

## ⚡ Why use it
- Simple setup, no complex configuration
- Automatically logs every moderation action
- Helps staff manage large communities efficiently
- Designed for stable and consistent moderation workflows

## 🧠 Built for
Servers that need structured moderation, clear logs, and reliable staff tools without extra hassle.
